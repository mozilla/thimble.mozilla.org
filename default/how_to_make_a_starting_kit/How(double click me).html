Creating Starting Kit Tutorial

I just create a Starting Kit, and I believe a good tutorial will help a lot of people to create more Starting Kit much easier.

  1. Create a new project and Publish it

    You first have to complete a project and publish it on thimble.

    All Starting Kits are projects that is published by someone.

    For example: Keep Calm Poster

    Tip: Projects on the homepage are hosted on thimbleprojects.org/mozillalearning, BUT we do NOT have to do that.

    Just host the page on your account and DO NOT Delete it.

  2. Add tutorial.html

    Add a tutorial.html directly into the file tree of your project.

    Thimble will automatically add the "TUTORIAL" button for you between the "PREVIEW" button and the reload button.

    Don't forgot to update your published project.

    The template "tutorial.html" is given in the same folder as this file. :)

  3. Create a Teaching Kit and Publish it.

    This one is optional.(because I saw a lot of people didn't do that...)

    However, if you want your Starting Kit show up on the homepage, you need a Teaching Kit.

    I friendly modify the current two Teaching Kits. You can simply remix and publish it. :)

    Teaching Kit should published as a different project.

    Templates: Template 1, Template 2

  4. Add Image.

    Every Starting Kit has a cover image.

    The size of image should be: 271 * 278 !

    There are two ways to add image:

    1. Add the image directly into the file tree of your project. (please name it as: thumbnail.???)
    2. Add the image under public\resources\img. (and name it as: thumbnail-<project name>.???)
  5. Create a pull request. Ask admin to change this file

    This file: "https://mozilla.github.io/thimble-homepage-gallery/activities.json" is the file that store the location Info of all Starting Kits.

    The format of new Starting Kit is:

    ,

    {

    "title" : "Name of your Starting Kit",

    "url" : "https://thimbleprojects.org/mozillalearning/your-project-id/",

    1. "thumbnail_url" : "https://thimbleprojects.org/mozillalearning/your-project-id/img-name",
    2. "thumbnail_url" : "https://thimble.mozilla.org/img/img-name",

    "description" : "Desc of you project",

    "tags" : ["tag1", "tag2", "tag3"],

    "author" : "mozilla",

    "author_url" : "http://learning.mozilla.org"

    }

    thumbnail_url depend on which way you host your cover img.

Great job, you finished the tutorial! You can make your own tutorials for others to follow by adding a tutorial.html page to any Thimble project.

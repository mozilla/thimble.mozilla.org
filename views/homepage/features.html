Add and manage files

Thimble works like a real web project, you can create, raname, delete and even upload your own files including folders. Makes it easy to work with your own images and photos. Or to open your old projects.

  • Download a .zip of your entire project
  • Drag & drop files like photos from your computer into Thimble

Turn the lights on and off

Thimble comes with a dark and light theme, so pick what's best for you. You can also change other options like the size of the text.

Follow & create embedded tutorials

Projects can have tutorials that help you (or your students) learn as they make changes to the projects the Remix.

Get code hints as you type

Don't get stuck trying to remember a lot of rules, Thimble will help you along the way for HTML, CSS and even JavasScript files. It's great. Add more detail here

Edit CSS right inÂ your HTML pages

Write CSS styles for your elements without switching away from the HTML page that you are editing. Put the text cursor on any element tag and hit Cmd/Ctrl + E to start writing CSS styles.

Publish projects to the web fast

You can publish your projects to the web with only a single click. Later on, when you make changes, you can easily update your project.

Pick colors right in the editor

Put the text cursor on any color value in a CSS file and press Cmd/Ctrl + E to open the colorpicker.

Test your site on a mobile device

Switching to the mobile preivew mode lets you see how your project would look on a mobile phone. You can also turn on a full-screen preview to get the big picture.

Find HTML elements fast

Turn on the DOM Inspector and hover over different elements in the preview. Thimble will highlight the related code in the HTML editor.

Image Filters

Add some fun to your project by adding filters to your images.
